Four specialized agents enable modular intelligence

Analyst Agent
Intelligence Generation
Runs core intelligence modules (SQPE, TIE, NDS) to generate probability estimates and identify value opportunities for each race.
Risk Agent
Capital Protection
Implements Portfolio Kelly Criterion for optimal bet sizing and enforces circuit breakers (max 3 losses, 20% daily limit) to protect bankroll.
Execution Agent
Market Interface
Connects to Betfair API for live bet placement, monitors execution quality, handles errors, and logs all transactions for audit trail.
Learning Agent
Continuous Improvement
Evaluates prediction accuracy post-race, tracks module-specific performance, and triggers model retraining when performance degrades.
