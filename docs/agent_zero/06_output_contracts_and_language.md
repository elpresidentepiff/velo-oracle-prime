# Output Contracts & Language

## Output Discipline

### Core Principle
Every output must contain **evidence and structure only**. No hype, no "just trust."

## Minimum Output Specification

### Required Elements (Every Race)

#### 1. Race Context + Type Classification
```
[Race Name] - [Date] - [Track] - [Going]
Type: [Flat/AW/NH] - [Subtype: Handicap/Maiden/etc]
Distance: [X furlongs/miles]
Field Size: [X runners]
```

#### 2. Pace Shape + Scenario Probabilities
```
Pace Scenario: [S2/S3/S4/S8]
Probability: [X%]
Early Leaders: [Horse A, Horse B]
Pressure Points: [Who contests whom]
Closers' Chance: [High/Medium/Low]
```

#### 3. Intent Signals (If Any)
```
Intent Detected: [Yes/No]
Signals: [List of signals]
- Signal 1: [Description]
- Signal 2: [Description]
Why They Matter: [Impact on race]
```

#### 4. Bias/Conditions Relevance
```
Track Bias: [None/Light/Strong]
Conditional Analysis: [How it affects race]
Going Impact: [Significant/Moderate/Minimal]
Draw Analysis: [Conditional, not absolute]
```

#### 5. Market Behaviour Read
```
Market Type: [Organic/Informed/False Steam/Resilient]
Liquidity: [High/Medium/Low]
Price Action: [Drift type, spread, depth]
Matched Volume: [X% of field size]
```

#### 6. Threats List
```
Threats to Model:
- Threat 1: [What can break it]
- Threat 2: [What can break it]
- Threat 3: [What can break it]
Risk Band: [Low/Medium/High]
```

## Language Rules

### Prohibited Language
- ❌ "Just trust"
- ❌ "Guaranteed"
- ❌ "Sure thing"
- ❌ "Hot tip"
- ❌ "Must win"
- ❌ "Lock"
- ❌ "Can't lose"

### Required Language
- ✅ "Scenario likelihood"
- ✅ "Risk band"
- ✅ "Evidence suggests"
- ✅ "Conditional on"
- ✅ "Probability"
- ✅ "Threats include"
- ✅ "Market indicates"

### Tone
- **Professional** - Not casual
- **Analytical** - Not emotional
- **Precise** - Not vague
- **Humble** - Not arrogant

## Quality Standards

### Minimum Requirements
1. **All 6 elements** present
2. **Evidence** for every claim
3. **Probabilities** not certainties
4. **Threats** identified
5. **No hype language**

### Quality Checklist
- [ ] Race context clear
- [ ] Pace scenario defined
- [ ] Intent signals (if any) listed
- [ ] Bias/conditions analyzed
- [ ] Market read included
- [ ] Threats identified
- [ ] Risk band stated
- [ ] No prohibited language
